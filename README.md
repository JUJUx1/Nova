# 🌌 Nova Player
**The Ultimate Terminal Music Experience for Termux.**

Nova Player is a high-performance, feature-rich CLI music player designed specifically for the Android/Termux environment. It combines a sleek `curses`-based UI with powerful background capabilities like YouTube downloading, lyrics synchronization, and audio visualization.
